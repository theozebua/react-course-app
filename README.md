# React Course App (Landing Page Template)

This is a test project built with:

- React JS
- Tailwind CSS

## Run Locally

Clone the project

```bash
  git clone https://github.com/theozebua/react-course-app.git
```

Go to the project directory

```bash
  cd react-course-app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
